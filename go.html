2. Hello World!

So... how you can start coding?, its easy!, jsut follow this steps

  • Make a folder : use "mkdir Golang-Course"
  • Enter to the folder : use "cd Golang-Course"
  • Create a new module with the next command : "go mod init Golang-Course". Obviusly you can change the name for other projects
  • Verify all is good ; tipe "ls" too see files, or if you are on windows, type "dir"

Now, lets start the coding!

In the Golang-Course foulder, create a new file with the name "main.go"

Now, you need to put this code



package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

So... what the heck is this?. first, the package main tells go that is the main executable.

import "fmt", fmt is like a library which give us functions like Println.

To declare the main function as you can see, we use "func main()"

So. fmt.Println its calling the function "println" from the fmt package, when you call a function you need to put () at the end. Print needs a argument, which is the text to print, so inside the () we put the string we want, in our case, "Hello World"

Now, try on your terminal, "go run main.go". You should se a good output. Congratulations!

3. Variables

How can I declare vars?

First, you need to know the data-types, which are the next

  • Integer Numbers : int
  • Floating Point Numbers : float32 & float64, flaot64 is recommended for more complex math
  • Booleans : True and False

GLOBAL VARS

A global variable is a variable which can be used on all code

For example, you can use in diferent functions

Here are some examples of declaring global variables


// Declare an integer variable for the year
var year int = 2025

// Declare a string variable for the month
var month string = "February"

// Declare a float variable for Pi
var pi float32 = 3.141592653

// Declare a global boolean variable
var global bool = false
// Obviusly you can use any name you want!

You need to declare globals outside a function

LOCAL VARIABLES

For this ones, you declare it inside a function and you can only use it inside that function

For this, you dont need to specify the data type(except for things like float32/64)

Example of local variables


year := 2025
month := "march"
pi := float32(3.14)
global := false

EXAMPLES

global

import "fmt"

4. if, else, and switch

For this, you should know whats if and else for, they are conditionals in least words

IF AND ELSE

These are used to take desicions in base of conditions

An example of this could be


if pears == true {
	println("Pears are best")
} else if apples == false {
	println("APPLE SUCKS")
} else {
	println("Not true, not false?")
}

This was an example with booleans only for make it easy

But, if tou use booleans, you should do this instead


func main() {
	pear := true
	apple := false

    if pear { // we only write the variable without == true ONLY IF ITS BOOLEAN
	    println("PEARS YAY")
    } else if !apple { // If you want to do a == false, you should put ! before the var to indicate its false
        println("NOT APPLE SO PEAR YAY")
    } else {
		println("not true, not fake")
	    }
}

Switch is for when you have TOO MUCH CONDITIONS, so it makes look more clear


func main() {
    fruit := "pizza"

    switch fruit {
        	case "apple": // Case works like if
        		println("Its an apple!")
        	case "pear": // here case works like else if
        		println("Its a pear!")
        	case "orange":
        		println("Its an orange!")
        	case "banana":
        		println("Its a banana!")
        	default: // default is like else
        		println("Sir that isnt a fruit ._.")
        	}
        }

5. MODULAR CODE

So, in this chapter we will see how to have multiple files to modularize the code

First, copy this structure to your Golang-Course foulder(the one we are working on)


        ├── go.mod
        ├── main.go
        └── operations/
            └── operations.go

On operations.go we gona add basic math functions


package operations

func Add(a int, b int) int { // WAIT, WHY DOES IT HAVE AN INT OUTSIDE THE ()?: that is because thats the value we're gonna get back
    return a + b // it returns an int
}

func Multiply(a int, b int) int {
    return a * b
}

And on main, we're gonna call the function


package main

import "Golang-Course/operations" // WE use the name of our module

func main() {
    Additions := operations.Add(1, 1)
    multiplication := operations.Multiply(5, 5)
    println("1 + 1", Additions)
    println("5 * 5", multiplication)
}

When you run it, you should see it working fine!

6. STRUCTS

For this, you will need to put more attention than usual, ¿why?, this is more harder than everything we saw on this guide. The finall boss is here!

¿What the heck is a struct?

A struct is a structure that helps us to store multiple data under only a name. You can think of it as an "custom datatype" that constains variables you declared

¿HOW TO USE STRUCTS?

Imagine we are making a race videogame, so structs are useful here

Alright, here goes the example!


package main

import "fmt" // To use Printf. Structs work fine without fmt, but we need it to print.

// Struct declaration
type Car struct { // Not a rule, but struct names usually start with an uppercase letter
        Engine     string
        Speed      int
        Model      string
        Suspension int // Honestly, not sure what suspension exactly means, so this is like "how much it bounces"
        Year       int
        Price      int
}

func main() {
    var BMWi8 Car                   // Declare a variable BMWi8 of type Car
    BMWi8.Engine = "1499cc I3 turbo" // I don't know much about cars, this is from Google :P
    BMWi8.Speed = 250               // 250 KM/H
    BMWi8.Model = "BMW I8"
    BMWi8.Suspension = 2            // Made-up value
    BMWi8.Year = 2014
    BMWi8.Price = 640000            // 640k, according to Google :P

    fmt.Printf("Your car is a %s with a %s engine and a speed of %d KM/H, has a suspension of %d, was released in %d, and costs %d\n", BMWi8.Model, BMWi8.Engine, BMWi8.Speed, BMWi8.Suspension, BMWi8.Year, BMWi8.Price)

    var Tesla Car // Another example
	Tesla.Engine = "Electric Motor"
    Tesla.Speed = 332               // 332 KM/H
    Tesla.Model = "Tesla Model 3"
    Tesla.Suspension = 3            // Made-up value again
    Tesla.Year = 2016
    Tesla.Price = 500000            // 500k

    fmt.Printf("\nYour car is a %s with a %s engine and a speed of %d KM/H, has a suspension of %d, was released in %d, and costs %d\n", Tesla.Model, Tesla.Engine, Tesla.Speed, Tesla.Suspension, Tesla.Year, Tesla.Price)
}
